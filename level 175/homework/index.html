რა არის SQL?

  • SQL ნიშნავს სტრუქტურირებულ შეკითხვის ენას
  • SQL საშუალებას გაძლევთ წვდომა გქონდეთ მონაცემთა ბაზებზე და მანიპულირება მოახდინოთ მათზე.
  • SQL გახდა ამერიკის ეროვნული სტანდარტების ინსტიტუტის (ANSI) სტანდარტი 1986 წელს, ხოლო სტანდარტიზაციის საერთაშორისო ორგანიზაციის (ISO) სტანდარტი 1987 წელს.

რა შეუძლია SQL-ს?

  • SQL-ს შეუძლია მონაცემთა ბაზაზე მოთხოვნების შესრულება
  • SQL-ს შეუძლია მონაცემების მოძიება მონაცემთა ბაზიდან
  • SQL-ს შეუძლია მონაცემთა ბაზაში ჩანაწერების ჩასმა
  • SQL-ს შეუძლია მონაცემთა ბაზაში ჩანაწერების განახლება
  • SQL-ს შეუძლია მონაცემთა ბაზიდან ჩანაწერების წაშლა
  • SQL-ს შეუძლია ახალი მონაცემთა ბაზების შექმნა
  • SQL-ს შეუძლია მონაცემთა ბაზაში ახალი ცხრილების შექმნა
  • SQL-ს შეუძლია მონაცემთა ბაზაში შენახული პროცედურების შექმნა
  • SQL-ს შეუძლია მონაცემთა ბაზაში ხედების შექმნა
  • SQL-ს შეუძლია დააყენოს ნებართვები ცხრილებზე, პროცედურებსა და ხედებზე

NOSQL

NoSQL არის მოძრაობა, რომელიც ხელს უწყობს პროგრამული უზრუნველყოფის სისტემებს , სადაც მონაცემთა მდგრადობა ზოგადად ხასიათდება რელაციური მოდელის გამოუყენებლობით , რომელსაც ჩვეულებრივ იყენებენ ტრადიციული მონაცემთა ბაზები ( RDBMS ). ტერმინი სინამდვილეში ეხება SQL ენას , რომელიც რელაციურ მონაცემთა ბაზებში ყველაზე გავრცელებული მონაცემთა მოთხოვნის ენაა და აქ აღებულია, როგორც მთელი რელაციური პარადიგმის სიმბოლო.
