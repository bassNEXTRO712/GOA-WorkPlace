Programic Language

მოდით ვისაუბროთ კომპიუტერულ ენაზე დღეს დღეისობით ძალიან პოპულარულ და აქტიურ პროგრამირების ენებზე პროგრამირება არ არის მხოლოდ კოდის ხაზების წერა, პროგრამირება არის კრეატიულობის, პრობლემების გადაჭრის, ლოგიკისა და ინოვაციურობის დინამკიური, მუდმივად განვითარებადი სამყარო. პროგრამირების უნარი უმჯობესდება პრაქტიკითა და გამოცდილებით, მისი სწავლა დიდ დროსა და ენერგიას მოითხოვს. რა არის პროგრამირება?

    • Hyper Text Language

    ჰიპერტექსტის მარკირების ენა ( HTML ) არის სტანდარტული მარკირების ენა [ a ] დოკუმენტებისთვის, რომლებიც შექმნილია ვებ ბრაუზერში გამოსატანად . ის განსაზღვრავს ვებ კონტენტის შინაარსს და სტრუქტურას . მას ხშირად ეხმარება ისეთი ტექნოლოგიები, როგორიცაა Cascading Style Sheets (CSS) და სკრიპტის ენები, როგორიცაა JavaScript , პროგრამირების ენა. ვებ ბრაუზერები იღებენ HTML დოკუმენტებს ვებ სერვერიდან ან ლოკალური მეხსიერებიდან და გადააქვთ დოკუმენტები მულტიმედია ვებ გვერდებზე. HTML აღწერს ვებ გვერდის სტრუქტურას სემანტიკურად და თავდაპირველად მოიცავდა მის გარეგნობას. HTML ელემენტები არის HTML გვერდების სამშენებლო ბლოკები. HTML კონსტრუქციებით, სურათები და სხვა ობიექტები, როგორიცაა ინტერაქტიული ფორმები, შეიძლება იყოს ჩასმული რენდერ გვერდზე. HTML საშუალებას გაძლევთ შექმნათ სტრუქტურირებული დოკუმენტები ტექსტის სტრუქტურული სემანტიკის მითითებით , როგორიცაა სათაურები, აბზაცები, სიები, ბმულები , ციტატები და სხვა ელემენტები. HTML ელემენტები გამოიყოფა ტეგებით , რომლებიც იწერება კუთხის ფრჩხილების გამოყენებით . ტეგები, როგორიცაა და პირდაპირ შემოაქვს კონტენტი გვერდზე. სხვა ტეგები, როგორიცაა და გარს და გვაწვდიან ინფორმაციას დოკუმენტის ტექსტის შესახებ და შეიძლება შეიცავდეს ქვეელემენტის ტეგებს. ბრაუზერები არ აჩვენებენ HTML ტეგებს, მაგრამ იყენებენ მათ გვერდის შინაარსის ინტერპრეტაციისთვის. (img,input,p) HTML-ს შეუძლია სკრიპტირების ენაზე დაწერილი პროგრამების ჩასმა , როგორიცაა JavaScript , რაც გავლენას ახდენს ვებ გვერდების ქცევასა და შინაარსზე. CSS-ის ჩართვა განსაზღვრავს შინაარსის სახეს და განლაგებას. მსოფლიო ქსელის კონსორციუმი (W3C), HTML-ის ყოფილი შემსრულებელი და CSS სტანდარტების ამჟამინდელი შემსრულებელი, 1997 წლიდან ხელს უწყობს CSS- ის გამოყენებას აშკარა პრეზენტაციის HTML-ზე. [ 3 ] HTML-ის ფორმა, რომელიც ცნობილია როგორც HTML5 , გამოიყენება აჩვენეთ ვიდეო და აუდიო, ძირითადად ელემენტის გამოყენებით JavaScript-თან ერთად.

    Cascadian Style Sheet

    Cascading Style Sheets ( CSS ) არის სტილის ფურცლის ენა, რომელიც გამოიყენება მარკირების ენაზე დაწერილი დოკუმენტის პრეზენტაციისა და სტილის დასაზუსტებლად , როგორიცაა HTML ან XML (მათ შორის XML დიალექტები, როგორიცაა SVG , MathML ან XHTML ). [ 2 ] CSS არის მსოფლიო ქსელის ქვაკუთხედი ტექნოლოგია HTML და JavaScript- თან ერთად . [ 3 ] CSS შექმნილია იმისათვის , რომ უზრუნველყოს შინაარსისა და პრეზენტაციის გამიჯვნა , მათ შორის განლაგება , ფერები და შრიფტები . [ 4 ] ამ განცალკევებას შეუძლია გააუმჯობესოს კონტენტის ხელმისაწვდომობა , ვინაიდან კონტენტი შეიძლება დაიწეროს მისი პრეზენტაციის გარეშე; უზრუნველყოს მეტი მოქნილობა და კონტროლი პრეზენტაციის მახასიათებლების დაზუსტებაში; ჩართეთ მრავალ ვებ გვერდს გააზიაროთ ფორმატირება შესაბამისი CSS-ის ცალკე .css ფაილში მითითებით, რაც ამცირებს სტრუქტურულ შინაარსში სირთულეს და განმეორებას; და ჩართეთ .css ფაილის ქეშირება, რათა გაუმჯობესდეს გვერდის ჩატვირთვის სიჩქარე გვერდებს შორის, რომლებიც იზიარებენ ფაილს და მისი ფორმატირება. ფორმატირებისა და შინაარსის გამიჯვნა ასევე შესაძლებელს ხდის ერთი და იგივე მარკირების გვერდის სხვადასხვა სტილში წარმოდგენას სხვადასხვა რენდერის მეთოდისთვის, როგორიცაა ეკრანზე, ბეჭდვით, ხმით (მეტყველებაზე დაფუძნებული ბრაუზერის ან ეკრანის წამკითხველის საშუალებით ) და ბრაილის შრიფტით . ტაქტილური მოწყობილობები. CSS ასევე აქვს წესები ალტერნატიული ფორმატირებისთვის, თუ კონტენტზე წვდომა ხდება მობილური მოწყობილობიდან . [ 5 ]

    Javascript

    კომპიუტერულ მეცნიერებაში JavaScript არის მოვლენებზე ორიენტირებული მრავალ პარადიგმური პროგრამირების ენა , რომელიც გამოიყენება როგორც ვებ კლიენტის მხარეს , ასევე სერვერის პროგრამებში ( Node.js ) RESTful API-ების, დესკტოპის და ჩაშენებული აპლიკაციების, ვებსაიტების და ვებ აპლიკაციების შესაქმნელად . დინამიური ეფექტები ინტერაქტიული სკრიპტის ფუნქციების მეშვეობით , რომლებიც გამოწვეულია მოვლენებით, რომლებიც მორიგეობით გამოწვეულია მომხმარებლის მიერ გამოყენებულ ვებ გვერდზე სხვადასხვა გზით ( მაუსი , კლავიატურა , გვერდის ჩატვირთვა და ა.შ.). თავდაპირველად შემუშავებული იყო ბრენდან ეიხის მიერ Netscape Communications- ის მიერ, სახელწოდებით Mocha , მოგვიანებით კი LiveScript , მას მოგვიანებით ეწოდა " JavaScript " და ფორმალური გახდა Java ენის სინტაქსით უფრო ახლოს Sun Microsystems- ისგან (რომელიც 2010 წელს შეიძინა Oracle- სგან ). პირველად სტანდარტიზებული 1997 წელს ECMA-ს მიერ, სახელწოდებით ECMAScript , უახლესი სტანდარტი, 2022 წლის ივნისისთვის, არის ECMA-262 გამოცემა 13 [ 1 ] და ასევე არის ISO სტანდარტი (ISO/IEC 16262).
